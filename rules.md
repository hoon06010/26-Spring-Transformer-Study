# 출석 및 참여 규칙

주니어 트랙에서는 원활한 스터디 진행을 위해, 출석과 참여는 필수입니다. 아래 규칙을 참고해주세요.

## 기본 원칙
팀 프로젝트로 진행되는 만큼, **출석과 참여도가 굉장히 중요**합니다.

## 출석 관리
- 출석 원칙은 **팀장 전결제**로 하며, 판단 권한은 전적으로 팀장에게 있습니다.
- 팀장은 매주 활동 결과물과 출석 명단을 제출해야 합니다.
- 부당한 처리 발생 시 **디스코드 신문고**로 제보해주세요.

## 결석 처리
- **무단 결석 1회 발생 시 활동 중지 처리**됩니다.
- 해당 주차 결과물 미제출 시 무단 결석으로 간주됩니다.

## 공결 처리
- 공결 시 몸이 못 오면 **기여로 인정되는 활동**을 제출해야 합니다.

### 4주 코스
| 주차 | 대체 과제 예시 |
|------|----------------|
| 1주차 | 배포된 요약본 정독하고 A4 1장 분량으로 자필 요약 인증 |
| 2주차 | 팀 완성 코드를 보고, 주요 라인마다 상세 주석 10개 이상 달아 제출  |
| 3주차 | 팀 실험 결과(리그전)를 분석하여 성능 변화 원인/개선점 코멘트 5줄 이상 작성 |
| 4주차 | 이번 달 활동 회고록(배운점, 아쉬운점) 500자 이상 작성 |
